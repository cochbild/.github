# Support

## Getting Help

If you need help with a project, here's how to get support:

### Check Existing Resources

1. **README** — Start with the project's README for setup and usage instructions
2. **Issues** — Search [existing issues](https://github.com/cochbild) to see if your question has been answered
3. **Discussions** — Check the Discussions tab if the project has one enabled

### Ask a Question

- **Open an Issue** — Use the appropriate issue template for bugs or feature requests
- **Start a Discussion** — For general questions, ideas, or help requests

### What to Include

When asking for help, please provide:

- What you're trying to do
- What you've already tried
- Relevant error messages or logs
- Your environment (OS, language version, etc.)

## Security Issues

**Do not report security vulnerabilities through public issues.** See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

## Response Time

This is maintained as a personal/side project. I'll do my best to respond in a timely manner, but please be patient.
